# Ecosistemanet — Memoria del Negocio

_Última actualización: 2026-04-05_

---

## 🎯 Esencia

**Qué es:** Suite modular de infraestructura digital para gestionar ecosistemas de emprendimiento e innovación.

**Propuesta de valor core:** No vende software — vende **reducción de incertidumbre en ecosistemas complejos**.

**Rol de Jaime:** Cofundador (diciembre 2025 – presente)

**Estado:** Startup activa, fase inicial de construcción/comercialización.

---

## 🧩 Módulos (8)

| Módulo | Función | Claridad que entrega |
|--------|---------|----------------------|
| **Atlas** | Mapear y visualizar actores y relaciones del ecosistema | Claridad estructural |
| **Pulse** | Levantar datos y construir diagnósticos | Claridad diagnóstica |
| **Nexus** | Votaciones y toma de decisiones trazables | Claridad de gobernanza |
| **Focus** | Convertir conversaciones en información estructurada (IA) | Claridad conversacional |
| **Vantage** | Evaluar proyectos con rúbricas + IA contextualizada | Claridad de decisión |
| **Metron** | Seguimiento de tiempo, tareas e hitos operativos | Claridad de ejecución |
| **Skopia** | Mapear y diagnosticar talento emprendedor | Claridad sobre talento |
| **Agora** | Gestionar eventos, inscripciones y participación | Claridad de interacción |

**Flujo principal:**  
Pulse → Focus → Atlas → Skopia → Vantage → Nexus → Metron → Agora → (retroalimenta todo)

---

## 👥 Clientes Objetivo

- **Ancla:** Gobiernos regionales, agencias de desarrollo, universidades, corporaciones de innovación
- **Modulares:** Incubadoras, aceleradoras, hubs, consultoras, redes de emprendimiento
- **Operativos:** Organizaciones que ejecutan convocatorias, administran comités, necesitan trazabilidad

---

## 💰 Modelo de Negocio

**Enfoque:** Consultivo-modular — el cliente entra por un problema concreto, luego expande.

**Estructura de ingresos:**
1. **Implementación inicial** — configuración, carga de datos, modelado
2. **Suscripción SaaS** — acceso por usuario/organización
3. **Servicios premium** — consultoría analítica, operación asistida, acompañamiento de gobernanza
4. **Consumo variable** — créditos en Focus y Vantage (por volumen de análisis)

---

## 🏗️ Arquitectura Técnica (capas)

1. **Frontend** — Aplicación web modular (analítica, admin, evaluación, gobernanza, operación)
2. **Backend** — Servicios, workflows, reglas, permisos, scoring, trazabilidad
3. **Datos** — Base relacional + almacenamiento documental + búsqueda semántica
4. **Inteligencia** — Embeddings, RAG, scoring contextual, síntesis, clasificación
5. **Infraestructura** — Auth, storage, observabilidad, seguridad, colas

**IA prioritaria en:** Focus (transcripción/síntesis), Vantage (scoring contextual), búsqueda semántica, detección de patrones.

## 🚀 Roadmap Evolutivo

| Etapa | Descripción |
|-------|-------------|
| **Etapa 1 (actual)** | Plataforma estructurada: captura, organiza, evalúa, decide, mide |
| **Etapa 2** | Plataforma inteligente: interpreta, recomienda, alerta, prioriza con IA |
| **Etapa 3** | Plataforma semiautónoma: detecta señales, propone acciones, aprende de resultados |

**Módulos futuros potenciales:** Observatorio (capa pública de indicadores) · Recomendador · Portafolio · Impacto longitudinal · Copiloto EcosistemaNet

---

## 🔑 Diferenciación

1. Modela complejidad real (actores, relaciones, decisiones, conversaciones, talento, ejecución)
2. Conecta capas normalmente separadas (diagnóstico + gobernanza + evaluación + operación)
3. Propuesta de claridad, no de feature list
4. Mezcla software con método (especialmente Atlas y Vantage)

**Barreras de entrada:**
- Metodológica (pocos saben modelar ecosistemas complejos)
- De datos (acumulación de tipologías, plantillas, históricos)
- Funcional (competidores resuelven fragmentos; EcosistemaNet conecta todo)
- Institucional (costo de reemplazo sube al instalar gobernanza interna)
